# 🛡️ 안전한 비밀번호 관리자
30904 김도현 30911 백현아<br>
암호화적으로 강력한 방법을 활용해 비밀번호를 생성하고, 입력한 비밀번호의 보안 강도를 간단히 확인할 수 있는 웹 기반 프로그램입니다.

## 소개

이 프로그램은 사용자가 직접 브라우저에서 비밀번호를 생성하고 관리할 수 있도록 만든 간단한 비밀번호 생성기입니다.  

별도의 서버 저장 없이 클라이언트 측에서 즉시 동작하도록 구성되어 있으며, 직관적인 UI를 통해 다음 기능을 제공합니다.

- 안전한 비밀번호 생성

- 비밀번호 길이 조절

- 대문자 포함 여부 선택

- 특수문자 포함 여부 선택

- 혼동하기 쉬운 문자 제외

- 생성된 비밀번호 복사

- 비밀번호 보안 강도 확인

## 주요 기능

### 1. 비밀번호 생성기

사용자가 원하는 조건에 맞춰 비밀번호를 생성할 수 있습니다.

- 비밀번호 길이: 4자 ~ 50자

- 대문자 포함 가능

- 특수문자 포함 가능

- 혼동하기 쉬운 문자 제외 가능  

  - 예: `i, I, l, 1, O, 0`

### 2. 복사 기능

생성된 비밀번호를 클립보드에 복사할 수 있습니다.

- 복사 성공 시 아이콘이 `✅`로 변경되어 사용자에게 피드백 제공

- 잠시 후 다시 원래 복사 아이콘으로 복구

### 3. 생성 기록 표시

최근 생성된 비밀번호의 생성 시각과 일부 내용을 간단히 표시합니다.

### 4. 보안성 확인

사용자가 입력한 비밀번호의 길이에 따라 보안 강도를 간단히 판별합니다.

- 8자 미만: 위험

- 8자 이상 12자 미만: 보통

- 12자 이상: 강력

### 5. 정보 탭

프로그램의 목적과 저장 방식, 버전 정보를 확인할 수 있습니다.

## 화면 구성

프로그램은 3개의 탭으로 구성되어 있습니다.

- 생성기: 비밀번호 생성 및 옵션 설정

- 보안성 확인: 비밀번호 강도 확인

- 정보: 프로그램 소개 및 버전 정보

## 사용 기술

- HTML5: 웹 페이지 구조 구성

- CSS3: UI 디자인 및 레이아웃 구성

- JavaScript: 탭 전환, 비밀번호 생성, 복사, 강도 확인 기능 구현

## 동작 방식

이 프로그램은 브라우저에서 직접 실행되며, 서버와 통신하지 않습니다.  

즉, 생성된 비밀번호나 사용자가 입력한 비밀번호는 외부 서버에 저장되지 않습니다.

## 실행 방법

1. 프로젝트 파일을 다운로드합니다.

2. HTML 파일을 브라우저에서 엽니다.

3. 바로 비밀번호 생성 및 보안성 확인 기능을 사용할 수 있습니다.
